CALCULATION ENGINE — VALIDATED FORMULAS (DO NOT MODIFY)
=================================================================

1. Handicap Differential (per round):
   diff = (score - rating) × (113 / slope)

2. Estimated Handicap:
   Sort all diffs ascending → take best (lowest) up to 8 → average
   estHcp = sortedDiffs.slice(0,8).reduce((s,d)=>s+d,0) / min(8, n)

3. Tier Assignment:
   Find tier whose representative HCP is closest to estHcp

4. Stat Normalization:
   userFW    = totalFwHit / (18 - par3s per round, summed)
   userGIR   = totalGIR / totalHoles
   userPutts = totalPutts / totalGIR
   userScram = totalScrambles / totalMissedGreens

5. Target Ratios:
   ratioFW    = max(0, min(100, 100 × userFW / target.fw))
   ratioGIR   = max(0, min(100, 100 × userGIR / target.gir))
   ratioPutts = max(0, min(100, 100 × target.putts / userPutts)) ← INVERTED
   ratioScram = max(0, min(100, 100 × userScram / target.scramble))

6. Strokes by Category:
   rawD = -3.5 × (max(0, 100 - ratioFW)    / 100)
   rawA = -6.0 × (max(0, 100 - ratioGIR)   / 100)
   rawP = -2.0 × (max(0, 100 - ratioPutts) / 100)
   rawS = -3.5 × (max(0, 100 - ratioScram) / 100)
   scaleFactor = totalGap / (|rawD|+|rawA|+|rawP|+|rawS|)
   strokesX = |rawX| × scaleFactor

─────────────────────────────────────────────────────

New structure: 3 tabs inside the Practice Plan card
  · Tab 1: My Plan
  · Tab 2: Warm Up
  · Tab 3: My Log

MY PLAN TAB:
Reads: settings.practiceTime + d.breakdown (priority order)
Renders: session cards (Session 1, Session 2, etc.)
Each session card contains drill cards.

Session structure by time:
  30 min  → 1 session: 1 improvement drill — #1 priority
  1 hr    → 1 session: 2 improvement drills #1 + 1 drill #2
  1.5 hrs → 2 sessions: S1: 2 drills #1 · S2: 1 drill #2 + 1 drill #3
  2 hrs   → 2 sessions: S1: 2 drills #1 + 1 drill #2 · S2: 2 drills #2 + 1 maintenance #3/#4
  3 hrs   → 3 sessions: S1: #1 full · S2: #2 full · S3: #3 + maintenance #4
  4 hrs   → 4 sessions: S1-S3 each category full · S4: Rotation (label only, no specific drill)

Drill card contains:
  · Drill name
  · Type badge: IMPROVEMENT (green) or MAINTENANCE (grey)
  · Sets × reps (e.g. "3 sets of 10 balls")
  · Description (1-2 sentences, specific)
  · Success criteria (e.g. "land 7 of 10 within 6 feet")
  · YouTube link (opens in new tab)
  · "Log This Session" button → opens My Log entry form

DRILL LIBRARY (static, per category):
Build a DRILLS object with keys: 'Approach Shots', 'Short Game', 'Driving', 'Putting'
Each has: improvement[] and maintenance[] arrays
Minimum 3 improvement drills + 2 maintenance drills per category
YouTube links: use credible instructors (Rick Shiels, MrShortGame, Mark Crossfield)

STEP 3: Warm Up tab
─────────────────────────────────────────────────────
Always visible regardless of calculation state.
Tied to d.breakdown[0].category (#1 priority).
Version shown driven by settings.warmupTime.

Format: numbered step list, specific and sequential.
10-min version: parking lot friendly, no range required.
20/30/45-min versions: range-based, progressively more thorough.

4 warmup routines per category × 4 time lengths = 16 total routines.

             category: 'Approach Shots', result: 14, total: 20,
             date: '2026-03-05T14:30:00Z', sessionNum: 1 }

UI:

      - Drill name (pre-filled, read-only)
      - Result: [number] of [total] (e.g. 14 of 20)
      - Date (auto-filled, editable)
      - Submit → saves to log, triggers badge check

STEP 5: Badges
─────────────────────────────────────────────────────
Displayed as a badge shelf below the Practice Plan card.
Earned: full color + name + date earned.
Unearned: greyed out + requirement shown.

COMPLETION BADGES (read from log entry count):
  🥉 First Rep       — 1 session logged
  🥈 On A Roll       — 5 sessions logged
  🥇 Committed       — 20 sessions logged

IMPROVEMENT BADGES (read from log, per drill):
  📈 Getting Sharper — any drill score improves vs first attempt
  🎯 Sharpshooter    — same drill improves 3 sessions in a row

TIER MILESTONE BADGES (read from calculation results):

  Store last tier in truestrokes_settings.lastTier.
  Badges: Welcome to 90s 🏌️ / Welcome to 80s / Welcome to Low 80s /
          Welcome to Single Digit / Welcome to Low Single Digit /
          Welcome to Scratch / Welcome to Plus 🏆

Badge check function: checkBadges(logEntries, currentKey)
  Called after every log save and after every Calculate.
  Saves earned badges to truestrokes_settings.badges = []
